# Code of Conduct

This project aims to be practical, respectful, and focused on technical collaboration.

## Expected Behavior

- Be respectful and direct.
- Keep criticism focused on code, design, tests, and user impact.
- Assume good intent, but ask for clarity when needed.
- Avoid sharing private data or credentials.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Publishing private information
- Deliberately disruptive behavior
- Repeated off-topic arguments after maintainers redirect the discussion

Maintainers may remove comments, close issues, or restrict participation when behavior makes collaboration unsafe or unproductive.
